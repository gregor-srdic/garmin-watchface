# Forerunner Watch Face — Connect IQ (Monkey C)

A 454×454 round AMOLED watch face mirroring the HTML design:
- 4 colored arc gauges around the rim
  - **red** — heart rate
  - **orange** — stress
  - **blue** — body battery
  - **rose** — calories
- 24-hour time, multi-color digits (orange → rose → purple → blue)
- Date + weather row just below the time

Targets: Forerunner 965, Fenix 7 Pro, Venu 3 / 3S, Epix 2 Pro 47/51mm.

## Project layout

```
monkeyc/
  manifest.xml               Connect IQ app manifest
  monkey.jungle              Build config
  source/
    ForerunnerWatchFaceApp.mc   AppBase entry point
    ForerunnerWatchFaceView.mc  WatchFace render code
  resources/
    strings/strings.xml      App name
    drawables/drawables.xml  Launcher icon registration
    images/                  (drop a 40x40 launcher_icon.png here)
```

## Building

1. Install the **Garmin Connect IQ SDK** and the VS Code "Monkey C" extension.
2. Open the `monkeyc/` folder as the project root in VS Code.
3. Run **Monkey C: Build for Device** and pick a target (e.g. `fr965`).
4. Sideload the resulting `.prg` to your watch via the Connect IQ desktop app, or run it in the simulator.

## Notes on fidelity vs. the HTML design

| HTML feature | Monkey C approach |
|---|---|
| Smooth anti-aliased arcs | `dc.setAntiAlias(true)` (AMOLED devices) + filled circles for rounded endcaps |
| CSS linear-gradient time text | Per-digit colors (orange / rose / purple / blue) — closest practical Monkey C equivalent |
| Glow under arcs | Omitted — would require multiple offscreen buffers and hurts AMOLED battery |
| `oklch()` colors | Approximated to the closest sRGB hex |
| Sun glyph (☀) | Inline unicode in the date row; swap for a `Bitmap` resource for a richer icon |
| Weather temp | Pulled from `Toybox.Weather.getCurrentConditions()`; respects system C/F |
| Stress/Body/etc values | Coerced to integers via `.toNumber()` and `.format("%d")` to avoid `35.000000` rendering |

## Replace before publishing

- `resources/images/launcher_icon.png` — must be a real 40×40 PNG.
- `manifest.xml` `id="..."` — replace with a fresh UUID generated by the SDK (`monkeyc -g`).
- App store name / description / screenshots.
